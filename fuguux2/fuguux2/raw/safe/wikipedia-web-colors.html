Web colors

Hex Triplet

A hex triplet is a six-digit (or eight-digit), three-byte (or four-byte) hexadecimal number used in HTML, CSS, and SVG to represent a color. Each pair of digits represents red, green, and blue channel intensity (00–FF), producing 16,777,216 possible colors (2²⁴). A shorthand three-digit form exists, reducing the palette to 4,096 colors. Leading zeros are required (e.g., 04, 08, 0F). The eight-digit form adds an alpha (transparency) byte.

sRGB is the standard color space for web colors. One byte per component represents 0–255 intensity. Colors outside sRGB require the CSS RGB() function with values exceeding 100%.

HTML Color Names

Basic 16 colors (HTML 4.01, 1999)

white, silver, gray, black, red, maroon, yellow, olive, lime, green, aqua, teal, blue, navy, fuchsia, purple.

Extended colors

Derived from merging HTML 4.01, CSS 2.0, SVG 1.0, and CSS3 specifications. Approximately 124 additional named colors from the X11 color name set (originally from X Window System applications, first used in web browsers by early Mosaic and Netscape Navigator). CSS 2.1 added "orange" as a 17th basic color.

Color families: pink, red, orange, yellow, brown, purple/violet/magenta, blue, cyan, green, white, gray — each with named variants and specific hex/RGB values.

CSS Colors

CSS specifications define named colors and functional notation:

  • RGB: integer (0–255) or percentage values per channel.
  • RGBA: RGB plus an alpha channel for transparency.
  • HSL: hue (0–360°), saturation (0–100%), lightness (0–100%).
  • HSLA: HSL plus alpha transparency.
  • HWB: hue, whiteness, blackness (introduced in CSS Color Level 4).
  • The transparent keyword.
  • System colors (deprecated in CSS3): allowed styling based on the operating system's own UI color palette.

CSS Color Level 4 and Level 5

Level 4 introduces device-independent color spaces: CIE Lab, CIE LCH, OKLab, OKLCH, and XYZ. These allow wider-gamut colors beyond sRGB. Wide-gamut RGB profiles added: Display P3, ProPhoto, REC.2020, and Adobe RGB.

RebeccaPurple (#663399) was added to the CSS specification in 2014 as a memorial for Rebecca Meyer, daughter of CSS Working Group member Eric Meyer.

Level 5 (draft) includes the color-mix() function for blending two colors, and relative color syntax for adjusting one color relative to another.

Web-Safe Colors

In the mid-1990s, most monitors could display only 256 colors simultaneously. Colors outside a system's palette were dithered — rendered as a pattern of two nearby palette colors — producing a speckled texture rather than a solid fill. To avoid this, web designers used a standardized "web-safe" palette of 216 colors built from six equally-spaced hex values (00, 33, 66, 99, CC, FF) per RGB channel (6³ = 216).

Even this palette was an overcount: only 22 of the 216 colors are "really safe" on 16-bit displays — predominantly greens, yellows, and cyans. The article includes a full 216-color reference table organized by hex code.

By the early 2000s, 24-bit TrueColor displays had become standard, making web-safe colors practically obsolete.

Accessibility

Color selection

Both foreground (text) and background colors should be specified explicitly, rather than relying on browser defaults for one while setting the other. If only one is set, a user with a custom browser color scheme may end up with an unreadable combination — e.g., white text on a white background, or black text on a black background.

Color contrast

WCAG defines minimum contrast ratios between foreground and background:

  • 4.5:1 — minimum for normal-size text (WCAG AA, success criterion 1.4.3).
  • 3:1 — minimum for large text (WCAG AA).
  • 7:1 — enhanced contrast for normal text (WCAG AAA).

The article notes that dyslexic readers may benefit from contrast ratios below the maximum AAA recommendation — meaning very high contrast is not universally optimal and AAA should be understood as a ceiling to approach rather than a universal goal.

Historical Context

Early web browsers (Mosaic, Netscape Navigator) adopted X11 color names from X Window System desktop applications. By the 2000s, 24-bit TrueColor displays made web-safe colors obsolete, and the browser color-name repertoire was standardized across HTML, CSS, and SVG specifications.

Standards Bodies

W3C (CSS Working Group, SVG Working Group), WHATWG.
