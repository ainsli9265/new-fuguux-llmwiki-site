# Project Purpose

## Goal

 **A knowledge base of user experience (UX) design concepts that can be used to help describe why a given UX problem matters and present useful design solutions. It aims to document best practices, methodologies, and insights to foster a deeper understanding and consistent application of UX principles across projects. When ingesting a new source, update existing entity and concept pages to incorporate new information rather than creating duplicate pages.** 

## Scope

**In scope:**
Anything pertaining to UI/UX design  
* Usability testing techniques and reports
* Design principles and heuristics
* Information architecture patterns
* Wireframes, mockups, and prototypes (as examples of concepts)
* Accessibility guidelines

**Out of scope:**
* Specific code implementation
* Forewards and Acknowledgements
* Specific names like authors, editors
* Information not relevant or practical to user interface and user experience design
